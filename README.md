
# Shipthis Internal - gRPC-Protobuffers 

Protocol buffer for interal communication of shipthis microservices.

## Installation

Install protocol buffer library

#### Javascript
```bash
  npm install shipthisnx-protobufs
```

#### Python

```bash
  pip3 install shipthisnx-protobufs
```
    

  
## Supported Backends

- Python

- Rust

- Javascript
